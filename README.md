## Features

- Execute git pull command in certain remote sever from local.
- ...

## Requirements

- Python 2.7+ or 3.4+
- 3rd Package:
  - fabric
  - PyYaml

## Useage:

Change the configuration elements in conf.yaml file with your values.

Open your terminal:

```bash
python fabfile.py
```
